「一個 Gopher 一生中一定會在某個地方用到 Reflect」
        ——沒有人這麼說過。
Reflect 在 Go 之中扮演了不可或缺的角色
現實生活中的開發總是在靜態的好處與動態的彈性中尋求平衡
最常見的例子將 JSON 如此動態的資料對應在 Struct 中
這之中的關鍵便是 Reflect

在這個 Session 中

以及 Reflect 當中做了哪些實作
最後，也會分享一些使用 Reflect 的小技巧
